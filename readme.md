# 駄菓子屋さん 防犯カメラ＋分析アプリ

## ./server
  - Pythonアプリ
    - Flask
    - MediaPipe 姿勢（骨格）推定AI
    - ~~DeepFace~~ 年齢性別推定AI　性能低いのでやめ
    - InsightFace 年齢性別推定AI


## ./client
  - Reactアプリ（Vite）
    - TypeScript

## ./output
  - ./output/ 直下にデータベースを保存
  - ./output/video に動画を保存
  - ./output/thumbnail にサムネイル画像を保存
  - reactのフォルダの外にあるので、Flask経由で表示させる
  - 動画とサムネイル画像はYYYYMMDD-HHMMSS.mp4, YYYYMMDD-HHMMSS.jpgで同じファイル名
  - データベース SQLite3
    - cameraテーブル
      - datetime 日付 YYYY-MM-DD HH:MM:SSのフォーマットの文字列
      - filename ファイル名のベース YYYYMMDD-HHMMSSのフォーマットの文字列
      - duration 動画の長さ（秒）
      - gender AIが推定した性別
      - age AIが推定した年齢
      - ~~male_prob 男性の確率~~
      - ~~female_prob 女性の確率~~

## ./snapshot
  - 人を検出しなくても、毎時0分に画像を保存

## 基本機能
  - 管理者がログインし、カメラを起動させる
  - カメラが起動されていれば管理者も他クライアントもライブ映像を見ることができる
    - AIが人を検知すると録画開始
    - AIが人を検知しなくなって一定時間後に録画完了
    - 年齢は各フレームの画像で推定　最終結果は各フレームの平均値
    - 性別は各フレームの画像で推定　最終結果は多数決　拮抗したとき不明にするために両者の比率も保存
  - 各データは日付を選択して一覧表示
  - ビデオ再生機能、ダウンロード機能、データ削除機能
  - 分析画面　時間ごとの利用者、男女比、年齢別

## 全体的なしくみ
  - 起動は容易　キーボード、マウス、モニター不要　電源スイッチも不要
    - PC（Ubuntu）: BIOS設定で、コンセント通電だけで電源オン
    - サービスによりアプリ自動起動
  -  偶然、モニターがタッチ対応だったので、タッチ前提のデザインにする
    - タッチでシャットダウンを実装する
    - USBコネクタを1個ケースの外に出し、データ授受できるようにする
